File Naming Rules

✅ If you are assigned Question-1, upload your solution inside the Question-1 folder.

✅ If you are assigned Question-2, upload your solution inside the Question-2 folder.

✅ Do NOT create your own folder.

✅ Create a Python file using the following naming convention:

yourname_sol1.py

Example:

ram_sol1.py priya_sol1.py rohit_sol1.py
